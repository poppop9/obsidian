>[!hint] 什么是反射 ？
>反射允许对<u>成员变量</u>，<u>成员方法</u>，<u>构造方法</u>的信息进行编程访问

>[!hint] 反射有什么用 ？
>- 比如 idea 里的方法提示，当你输入一个类时，idea 会获取这个类中的方法，然后给你提示

# 获取
>获取的不是 Java 文件，而是 <u>Java 字节码文件</u>【`.class`】

## 获取 class 对象
- `Class.forname("全类名")` ==源代码阶段==使用
- `类名.class` ==加载阶段==使用
- `对象.getClass();` ==运行阶段==使用
