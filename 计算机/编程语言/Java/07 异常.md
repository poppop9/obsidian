# 异常处理
## try…catch…
```java
try {
	可能出现异常的代码
	……
} catch(异常类名，变量名) {
	异常处理的代码
	……
}
……
A
```

***执行流程：***
1. 执行 try 内的代码
	1. 未出现异常，不执行 catch 内的代码
	2. 出现异常
		1. 生成一个异常类对象，该对象被提交给 Java 运行时系统
		2. Java 运行时系统到 catch 中匹配异常类，匹配成功，则执行 catch 内代码
		3. 接着继续向下执行，直到代码 A

## throws
>`throws` 仅仅是抛出异常，***并没有做任何处理***，所以程序遇到异常还是会终止

```java
public void method() throws 异常名 {
	……
}
```

>[!hint] 在我们处理不了时，或偷懒时，我们会选择 `throws`
