>[!hint] 字符串的值在创建之后不能更改

# 创建字符串
- 通过 `char` 数组创建【创建同一段字符串时，地址不同】
```java
public static void main(String[] args) {
	char[] zifu = {97, 98, 99};
	String s1 = new String(zifu);
	String s2 = new String(zifu);
	System.out.println(s1 == s2);
}

---
false
```

- 直接创建【创建同一段字符串时，地址相同】
```java
public static void main(String[] args) {
	String s3 = "greenteck";
	String s4 = "greenteck";
	System.out.println(s3 == s4);
}

---
true
```
