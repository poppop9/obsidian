# 介绍
- ***图形界面***：Linux 发行版上看到的图形界面都只是运行在 Linux 系统之上的一套软件，跟 windows 完全不同

---

Linux 默认提供了 6 个纯命令行界面的终端让用户登录。在物理机系统上你可以通过使用`[Ctrl]`+`[Alt]`+`[F1]～[F6]`进行切换，当你切换到其中一个终端后想要切换回图形界面，你可以按下`[Ctrl]`+`[Alt]`+`[F7]`来完成
