# 环境变量
## 变量
>[!hint] 变量
>- 与其他编程语言相同，有类型，有作用域，可以给变量赋值
>- 变量名只能是英文字母、数字或者下划线，且不能以数字作为开头

- 创建变量 `declare 变量名`
- 给变量赋值 `tmp=shiyanlou` 
- 读取变量 `echo $变量名` 【~~`$` 表示引用一个变量的值~~】

```bash
$ declare testname                             
$ testname=ttt111                                                       
$ echo $testname                                           
ttt111
```
