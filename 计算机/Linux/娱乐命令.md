# 动物说话
```bash
# 更新软件包
sudo apt update
sudo apt install -y cowsay

cowsay hello shiyanlou

# 打印所有支持的动物
cowsay -l

# 使用'-f'选择动物种类
cowsay -f elephant hello shiyanlou
```
