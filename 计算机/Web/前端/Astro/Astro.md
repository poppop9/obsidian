
# ❤️ 项目结构
- `src/*` - 项目源代码（组件、页面、样式 ……）
- `public/*` - 非代码、未处理的资源（字体、图标 ……）
- `package.json` - 项目清单
- `astro.config.mjs` - Astro 配置文件
- `tsconfig.json` - TypeScript 配置文件

---

<u>`src/`</u>
- `src/components` 组件 ：可以创建 Astro ，React ，Vue …… 组件
- `src/content` 存储内容集合
- `src/layouts` 存放布局组件
- `src/pages` 页面
- `src/styles`  CSS 或 Sass 文件

<u>`public/`</u> ：存放不用在构建过程中处理的资源
