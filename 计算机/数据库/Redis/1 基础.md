>[!quote] Redis
>Redis 是非关系性数据库，是基于内存的数据存储系统，**所以性能非常高**

# 安装配置
- 使用 Docker 安装 
```
docker run --name redis -d -v /myredis/conf:/usr/local/etc/redis -p 6379:6379 redis
```

- 使用 Redis 图形化界面工具 Redis InSight 连接
	- 添加数据库
	- 输入主机地址，端口号

# 基本命令
Redis 用于数据库缓存，消息队列 ……

Redis 支持的数据结构：
- 基本数据类型
	- 字符串
	- 列表
	- 集合
	- 有序集合
	- 哈希
- 高级数据类型
	- 消息队列 Stream
	- 地理空间 Geospatial
	- HyperLogLog
	- 位图 Bitmap
	- 位域 Bitfield

## Set，Get
Redis 默认使用字符串来存储数据

- `SET 键名 值名` 设置键值对
- `GET 键名` 根据jian'ming

```bash
SET 
```
