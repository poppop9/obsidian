>[!quote] 大多数 SQL 支持以下函数：
>- 算术操作的数值函数
>- 生成美观的输出内容的格式化函数
>- 返回 DBMS 特殊信息的系统函数
>- 聚合函数

# 数值函数
>[!hint] 数值函数是各种 DBMS 最一致的函数

- **数学运算函数**
	- `CEIL(x)` 向上取整
	- `FLOOR(x)` 向下取整
	- `MOD(x, y)` 返回 x/y 的余数
	- `RAND()` 返回 0 到 1 之间的随机数
	- `ROUND(x, y)` 对参数 x 进行四舍五入，保留 y 位小数
- **数学函数与常量**
	- `ABS()` 返回一个数的绝对值
	- `COS()` 返回一个角度的余弦
	- `EXP()` 返回一个数的指数值
	- `PI()` 返回圆周率π的值
	- `SIN()` 返回一个角度的正弦
	- `SQRT()` 返回一个数的平方根
	- `TAN()` 返回一个角度的正切
