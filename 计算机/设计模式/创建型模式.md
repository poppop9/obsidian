# 工厂模式
- **未使用工厂模式**，~~使用 if … else … 结构~~
```java
public class Circle {
    public void Draw() { sout("Drawing a Circle"); }
}

public class Rectangle {
    public void Draw() { sout("Drawing a Rectangle"); }
}

public class Client {
    public void DrawShape(string shapeType) {
        if (shapeType == "Circle") {
            Circle circle = new Circle();
            circle.Draw();
        } else if (shapeType == "Rectangle") {
            Rectangle rectangle = new Rectangle();
            rectangle.Draw();
        } else {
            throw new ArgumentException("Invalid shape type");
        }
    }
}

// 使用
psvm {
	Client client = new Client();
	client.DrawShape("Circle");
	client.DrawShape("Rectangle");
}
```

## 简单工厂模式
```mermaid
graph LR
	subgraph 抽象产品
	    b[具体产品1]
	    c[具体产品2]
	    d[……]
    end

	e[使用者]--创建-->a
	a[工厂实体类]--生产-->抽象产品
```

```java
// 抽象产品
public interface IShape { void Draw(); }

// 具体产品类
public class Circle {
    public void Draw() { sout("Drawing a Circle"); }
}

public class Rectangle {
    public void Draw() { sout("Drawing a Rectangle"); }
}

// 工厂类
@Components
public class ShapeFactory {
	// 维护了一个map集合，代替了if……else……
    private Map<String, IShape> shapeMap = new HashMap<>();

    public ShapeFactory() {
        shapeMap.put("Circle", new Circle());
        shapeMap.put("Rectangle", new Rectangle());
    }

    public IShape CreateShape(String shapeType) {
        IShape shape = shapeMap.get(shapeType);
        if (shape == null) {
            throw new IllegalArgumentException("Invalid shape type");
        }
        return shape;
    }
}

// 客户端代码
public class Client {
	@Autowired
    private ShapeFactory shapeFactory;

    public void DrawShape(string shapeType) {
        IShape shape = shapeFactory.CreateShape(shapeType);
        shape.Draw();
    }
}

// 使用
Client client = new Client();
client.DrawShape("Circle");
client.DrawShape("Rectangle");
```

## 工厂方法模式
```mermaid
graph LR
	subgraph 抽象工厂
		b[工厂1]
		c[工厂2]
		d[工厂3]
    end

	a[使用者]--根据你要生产的产品，创建对应的工厂-->抽象工厂
	b--创建-->产品1
	c--创建-->产品2
	d--创建-->产品3
```

```java
// 抽象产品
public interface IShape { void Draw(); }

// 具体产品类
public class Circle implements IShape {
    public void Draw() { sout("Drawing a Circle"); }
}

public class Rectangle implements IShape {
    public void Draw() { sout("Drawing a Rectangle"); }
}

// 抽象工厂
public abstract class ShapeFactory {
    public abstract Shape createShape();
}

// 具体工厂
public class CircleFactory extends ShapeFactory {
    @Override
    public Shape createShape() { return new Circle(); }
}

public class RectangleFactory extends ShapeFactory {
    @Override
    public Shape createShape() { return new Rectangle(); }
}

// 客户端代码
public class Client {
    public void DrawShape(ShapeFactory shapeFactory) {
        IShape shape = shapeFactory.createShape();
        shape.Draw();
    }
}

// 使用
Client client1 = new Client();

client1.DrawShape(new CircleFactory());
client2.DrawShape(new RectangleFactory());
```

## 抽象工厂模式
```mermaid
graph LR
	a[使用者]--根据你要生产的产品，创建对应的工厂-->抽象工厂

	subgraph 抽象工厂
		b[工厂1]
		c[工厂2]
		d[工厂3]
    end

	subgraph 抽象工厂
    end

	b--创建-->产品1
	c--创建-->产品2
	d--创建-->产品3
```



1. **抽象工厂（Abstract Factory）**：定义了一个创建对象的接口，由子类决定实例化哪一个产品类。
    
2. **具体工厂（Concrete Factory）**：实现抽象工厂接口，生成一组相关的产品对象。
    
3. **抽象产品（Abstract Product）**：定义了产品的接口。
    
4. **具体产品（Concrete Product）**：实现抽象产品接口，由具体工厂实例化。
    
5. **客户端（Client）**：使用抽象工厂来创建产品，不直接依赖具体产品。
